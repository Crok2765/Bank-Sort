# Bank Sort

## 📖 Descripción

Bank Sort es un addon para World of Warcraft 3.3.5a diseñado para organizar automáticamente tus bolsas, banco personal y banco de hermandad.

El addon clasifica los objetos de forma automática para mantener una distribución más limpia y ordenada, facilitando la gestión de materiales, consumibles, equipamiento y recursos de hermandad.

> [!NOTE]
> Compatible con World of Warcraft 3.3.5a.

> [!TIP]
> Organiza tu banco con un solo clic y mantén todos tus objetos perfectamente ordenados.

---

## ✨ Características

* ✔ Ordenación automática de bolsas.
* ✔ Ordenación automática del banco personal.
* ✔ Ordenación automática del banco de hermandad.
* ✔ Reorganización inteligente de objetos.
* ✔ Notificación visual al finalizar el proceso.
* ✔ Compatible con la interfaz clásica.
* ✔ Integración visual con ElvUI.
* ✔ Fácil de usar.

---

## 📦 Funciones Principales

### 🎒 Bolsas

Organiza automáticamente todos los objetos almacenados en tus bolsas.

### 🏦 Banco Personal

Clasifica y reorganiza los objetos almacenados en tu banco.

### 👥 Banco de Hermandad

Permite ordenar pestañas del banco de hermandad para mantener una estructura limpia y organizada.

### 🔔 Notificaciones

Muestra un aviso cuando la ordenación ha finalizado correctamente.
